# Règle Opquast 144

> Les liens provoquant l'ouverture d'un logiciel externe ont un libellé explicite.

## Description

### Français
Les liens provoquant l'ouverture d'un logiciel externe ont un libellé explicite.

### English
Links which cause external software to open have an explicit label.

### Español
Los enlaces que hacen que se abran programas informáticos externos están explícitamente etiquetados.

---

## Objectifs

### Français
- Éviter sur le poste client l'ouverture inopinée d'un autre logiciel que le navigateur web.
- Améliorer l’accessibilité des contenus aux personnes handicapées

### English
- Avoid the unexpected launch of software other than the web browser on the client workstation.
- Improve the accessibility of content for people with disabilities.

### Español
- Prevenir la apertura inesperada de un programa informático que no sea el navegador web.
- Mejorar la accesibilidad del contenido para las personas con discapacidad

## Explication pédagogique

Le cas le plus fréquent est l’ouverture imprévue de votre logiciel de messagerie après un clic sur un lien « Contact ». Efforcez-vous de prévenir les utilisateurs.

---

## Solution recommandée

<p>Rédiger les libellés de liens concernés de manière à annoncer le comportement pour mieux l'anticiper. Par exemple, il est préférable de rédiger les liens <code>mailto</code> sous la forme «&nbsp;Envoyer un mail &nbsp;» plutôt qu'un simple lien «&nbsp;Contact&nbsp;».</p>
<p>En savoir plus: <a href="https://developer.mozilla.org/fr/docs/Web/HTML/Element/a">élément <code>a</code> sur MDN</a></p>

---

## Méthode de vérification

<p>Dans toutes les pages inspectées&nbsp;:</p><ul>
<li>Examiner le contenu pour y détecter les liens provoquant l'ouverture d'un logiciel externe (lien mail par exemple). </li><li>Pour chaque lien concerné, vérifier que le libellé du lien est suffisamment explicite pour que l'utilisateur soit prévenu de son comportement particulier.</li>
</ul>

---

## Vulgarisation

L’ouverture imprévue d’un logiciel tiers lorsque l’utilisateur pense être face à un lien classique de navigation est un facteur de désorientation immédiat.
